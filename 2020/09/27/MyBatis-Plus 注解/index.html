MyBatis-Plus 实体注解

https://mybatis.plus/

@TableName

属性 描述
value 表名
resultMap xml 结果映射 id

@TableId

属性 描述
value 主键字段名
type 主键类型（默认：idType.NONE，跟随全局）
主键类型 描述
AUTO 自增
ASSIGN_ID 雪花算法分配id，Long/Integer（ID_WORKER） 或 String（ID_WORKER_STR）通用
ASSIGN_UUID 32位 String（UUID），无排序

@TableField

属性 描述
value 字段名
el xml 中的映射字段（==有些许疑问，是映射的请求字段不再需要加#{}，还是映射resultMap中的字段==）
exist 是否存在数据库表中
insertStrategy NOT_NULL 非NULL判断
updateStrategy IGNORED 忽略判断
whereStrategy NOT_EMPTY 非空判断
fill 字段自动填充

@Version

乐观锁注解 ==乐观锁？==

@EnumValue

枚举

@TableLogic

逻辑删除注解，即假删除字段del_flag（是否删除：1-是；0-否）
